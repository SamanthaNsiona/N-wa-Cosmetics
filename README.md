# N-wa Cosmetics

Projet de développement d’une application web basée sur l’analyse faciale et la recommandation de soins cosmétiques personnalisés.  
L’objectif est d’utiliser l’intelligence artificielle pour analyser la forme du visage, estimer certains paramètres liés au type de peau et proposer des conseils beauté adaptés.

---

## 1. Objectifs du projet

- Détection faciale et analyse des traits (forme du visage, asymétrie, zones d’intérêt).
- Recommandation de soins cosmétiques et maquillage personnalisés.
- Intégration future d’une analyse colorimétrique (sous-tons du teint).
- Possibilité d’adapter la solution en version mobile.

---

## 2. Technologies utilisées

| Domaine | Technologies |
|--------|--------------|
| Front-end | HTML, CSS, JavaScript |
| Analyse d’image | TensorFlow.js, Face-api.js |
| Architecture | Application web responsive |
| Gestion de version | Git & GitHub |

---

## 3. Fonctionnalités principales

- Analyse faciale via webcam ou import d’image.
- Détection automatique de la forme du visage.
- Proposition de recommandations beauté adaptées.
- Interface web conçue pour être responsive.

---

## 4. Pistes d’amélioration

- Intégration d’une analyse colorimétrique plus avancée.
- Déploiement futur sur une application mobile (React Native ou Flutter).
- Mise en place de scénarios de test automatisés.
- Optimisation du design (UX/UI) et des performances.
